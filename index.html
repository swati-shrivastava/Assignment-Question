2. What is HTTP?

Ans: HTTP stands for HyperText Transfer Protocol. HTTP protocol is used for communication between web browser and server.
HTTP is a client-server protocol. For example The messages sent by the client, usually a Web browser, are called requests and the messages sent by the server as an answer are called responses.
HTTP is a TCP/IP based communication protocol, that is used to deliver data (HTML files, image files, query results, etc.) on the World Wide Web.
HTTP is used for send the request to the server and get the response from the server.
